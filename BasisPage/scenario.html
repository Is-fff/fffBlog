    # 大文件上传

    # 背景

    上传全屋方案XML

    # 问题

    网络断开之后，之前传的没了

    网络波动，啥都没了

    关机，传一半没法重传

    # 专业术语

    断点续传

    断开重连重传

    切片上传

    # 方案

    前端切片，将大文件切成一个个chunk,打上标识符（hash）和顺序index,后端根据index组合切片

    切完后，将切片log存储在localStorage/indexDb中，下次用户访问时查看是否有未完成的文件切片，有则继续上传

    # 反思

    切片过程，主线程卡顿（采用Web Worker多线程）
